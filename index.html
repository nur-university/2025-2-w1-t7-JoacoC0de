Juego Tetris Básico en Android Studio

Propósito: Este proyecto consistió en el desarrollo de una versión básica del clásico juego Tetris para dispositivos Android. El objetivo fue aplicar los conocimientos de programación orientada a objetos y el uso de interfaces gráficas en Android Studio, además de practicar la lógica de programación con figuras, rotaciones y detección de colisiones.

Funcionalidades: el juego permite mover y rotar piezas, acumular líneas eliminadas y aumentar la dificultad progresivamente. Incluye una interfaz sencilla y controles básicos para la jugabilidad en pantallas táctiles.

Tecnologías: Java, Android Studio, XML para interfaces gráficas. El código fuente se encuentra disponible en GitHub: Repositorio Tetris.

Aplicación de Contratación de Trabajadores

Propósito: Esta aplicación fue creada en Android Studio para conectar trabajadores independientes con empleadores que necesitan servicios puntuales. La idea es ofrecer una plataforma simple donde se pueda publicar una oferta laboral y los interesados postulen.

Funcionalidades: registro de usuarios, publicación de trabajos, listado de ofertas, contacto entre empleador y trabajador, e interfaz amigable adaptada a móviles.

Tecnologías: Java, Android Studio, SQLite para almacenamiento local. El código fuente se encuentra disponible en GitHub: Repositorio Trabajito.

Práctica de Redes y Subnetting

Propósito: Ejercicios para entender direccionamiento IPv4, máscaras, VLSM y diseño de subredes.

Resultados: Cálculo de subredes, verificación de rangos, identificación de direcciones de red y broadcast.

Herramientas: Markdown/HTML, Packet Tracer, hojas de cálculo y ejercicios manuales.
